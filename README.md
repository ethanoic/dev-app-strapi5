## MCP Server (PDF Content)

A Node.js-based Model Context Provider (MCP) server is available in the `mcp-server` directory.
This server reads PDF files from its `mcp-server/resource` subdirectory and exposes their combined textual content via an API endpoint (`/api/content`).

For more details on how to set up and run this server, please refer to the `mcp-server/README.md` file.
